# GitHub Audit

## Repo運用ルール
- リポジトリは最大3つまで
- 新機能の追加は api-biz-engine 内のフォルダで管理する

## リポジトリ分類
- A（アクティブ）: api-biz-engine
- B（凍結）: なし
- C（アーカイブ済み）: duo-llm-publisher, mac-agent, gpt-progress-bot
- D（削除候補）: auto-affiliate, -dropship-ai, -biz-template, -saas-optimizer

## Secrets管理
- 権限情報は GitHub Secrets で統一管理する
- env-secrets リポジトリは「削除予定」と明記
